Em geral

Quando você clica duas vezes no ícone WorldWideWeb, o aplicativo, por padrão, exibe sua própria página inicial, ~/WWW/default.html, ou, na falta dessa, a página inicial do sistema, /usr/local/lib/WWW/default.html.

A partir de então, se você quiser selecionar um documento referenciado por uma página que está sendo exibida, basta clicar duas vezes em uma área de texto cinza. (O cinza foi escolhido para não usar outras técnicas de realce como negrito ou itálico, que os autores podem querer usar para outras coisas. Quando a cor vem, podemos usar cor...)

Se você clicar duas vezes em qualquer outro lugar do documento, você selecionará uma palavra, como é usual no NeXT.

Algumas áreas cinzas levam a documentos inteiros, enquanto outras trazem um documento com uma área de destino específica realçada. Isso é chamado de âncora de destino. Não é cinza, a menos que leve a outro lugar, mas clicar duas vezes em uma âncora de destino seleciona a âncora inteira.

Índices

Certos documentos são índices. Eles normalmente têm apenas uma pequena quantidade de texto. Você não pode ler o índice sozinho (pode ser muito grande). Para acessar algo através do índice, use a opção de menu "Pesquisa por palavra-chave". A janela com o índice deve ser a janela principal (barra de título escura) quando você usa o painel de pesquisa de palavras-chave. As regras usadas pelos índices para pesquisar correspondências variam. Índices diferentes usam técnicas diferentes e cobrem diferentes faixas de informações.

Quando uma pesquisa é bem-sucedida, o resultado é uma página de referências a documentos. Você seleciona documentos a partir dele clicando duas vezes da maneira usual.

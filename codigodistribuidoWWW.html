Código distribuído WorldWideWeb

Veja os direitos autorais do CERN . Este é o arquivo README que você obtém quando desempacota um de nossos arquivos tar. Esses arquivos contêm informações sobre hipertexto, sistemas de hipertexto e o projeto WorldWideWeb. Se você tiver feito isso com um arquivo .tar, terá apenas um subconjunto dos arquivos.

ESTE ARQUIVO É UMA VERSÃO MUITO RESUMIDA DAS INFORMAÇÕES DISPONÍVEIS NA WEB. EM CASO DE DÚVIDA, LEIA A WEB DIRETAMENTE. Se você não tiver nenhum navegador instalado, faça isso por telnet para info.cern.ch (sem nome de usuário ou senha).

Estrutura do diretório de arquivo

Em /pub/www, além deste arquivo README, você encontrará os diretórios bin, src e doc. Os principais arquivos são os seguintes:

src/WWWLibrary_v.vv.tar.Z
A Biblioteca W3. Todas as fontes e Makefiles para sistemas selecionados.
src/WWWLineMode_v.vv.tar.Z
O navegador do modo de linha - todas as fontes e Makefiles para sistemas selecionados. Requer a Biblioteca.
src/WWW/LineModeDefaults_v.vv.tar.Z
A documentação on-line para o navegador de modo de linha, que você ainda pode desejar se obtiver o programa como binário. Um subconjunto da fonte completa acima.
src/WWWNextStepEditor.tar.Z
O navegador/editor de hipertexto para o NeXT -- fonte e binário.
src/WWWDaemon_v.vv.tar.Z
O daemon HTTP e os programas de gateway WWW-WAIS. Fonte.
doc/WWWBook.tar.Z
Um instantâneo de nossa documentação interna - preferimos que você acesse on-line - veja os avisos abaixo.
bin/xxx/bbbb
Binários executáveis ​​do programa bbbb para o sistema xxx. Verifique o que está lá antes de se preocupar em compilar. (Nota HP é chamado de "serpente")

Estrutura de diretório gerada

Os arquivos tar são todos projetados para serem desempacotados no mesmo (este) diretório. Eles criam diferentes partes de uma árvore de diretórios comum nesse diretório. Pode haver alguma duplicação. Eles também geram alguns arquivos neste diretório: README.*, Copyright.* e algumas instruções de instalação (.txt).

Navegador/Editor NeXTStep

O navegador para o NeXT são os arquivos contidos no diretório de aplicativos WWW/Next/Implementation/WorldWideWeb.app e são compilados. Ao instalar o aplicativo, você pode configurar a página padrão, WorldWideWeb.app/default.html. Estes devem apontar para algumas informações úteis! Você deve mantê-lo atualizado com indicadores de informações em seu site e em outros lugares. Se você usar a página inicial do CERN, observe que há um link na parte inferior para a cópia mestre em nosso servidor. Consulte as instruções de instalação.

Navegador de modo de linha

Binários disso para alguns sistemas estão disponíveis em /pub/www/bin/.

. Se o binário existir para o seu sistema, pegue-o e também o /pub/www/WWWLineModeDefaults.tar.Z. Desembrulhe a documentação e coloque (link) seu diretório em /usr/local/lib/WWW em sua máquina. Coloque o executável www em seu caminho em algum lugar e pronto.

Se não existir nenhum binário, proceda da seguinte forma. Pegue o arquivo tar de origem WWWLineMode_v.vv.tar.Z , descompacte e descompacte-o. Você encontrará o navegador de modo de linha em WWW/LineMode/Implementation/... (Consulte as notas de instalação ). Faça o mesmo (no mesmo diretório) com o arquivo WWWLibrary_v.cc.tar.Z para obter a biblioteca comum.br

Você terá um manual imprimível ASCII no arquivo WWW/LineMode/Defaults/line-mode-guide.txt que você pode imprimir nesta fase. Esta é uma cópia congelada de parte da documentação online.

Os subdiretórios para WWW/LineMode contêm Makefiles para sistemas para os quais já portamos. Se o seu sistema não estiver entre eles, crie um novo subdiretório com o nome do sistema e copie o Makefile de um existente. Altere os nomes dos diretórios conforme necessário. POR FAVOR, INFORME-NOS SOBRE AS ALTERAÇÕES QUANDO TER FEITO A PORTA. Esta é uma condição de seu uso deste código, e evitará que outros repitam seu trabalho e evitará que você o repita em versões futuras.

Ao instalar os navegadores, você deve configurar a página padrão. Este é /usr/local/lib/WWW/default.html para o navegador de modo de linha. Isso deve apontar para algumas informações úteis! Você deve mantê-lo atualizado com indicadores de informações em seu site e em outros lugares. Se você usar a página inicial do CERN, observe que há um link na parte inferior para a cópia mestre em nosso servidor.

Alguma documentação básica sobre o navegador é fornecida com a página inicial no diretório WWW/LineMode/Defaults. Um arquivo tar separado desse diretório (WWWLineModeDefaults.tar.Z) está disponível se você quiser apenas atualizá-lo.

O restante da documentação está em hipertexto e, portanto, poderá ser lido mais facilmente com um navegador. Sugerimos que após a instalação do navegador, você navegue pela documentação básica para estar ciente das opções e possibilidades de personalização, por exemplo.

Navegador Viola para X11

Viola é um aplicativo X11 para leitura de hipertexto global. Se um binário estiver disponível em sua máquina, em /pub/www/bin/.../viola*, pegue-o e também o arquivo tar "apps" do Viola que contém os scripts necessários.

Para gerar isso a partir do código-fonte, você precisará da biblioteca W3 e dos arquivos-fonte do Viola. Existe um Imakefile com o diretório fonte da viola. Você precisará gerar as bibliotecas XPA e XPM e a biblioteca W3 antes de fazer a própria viola.

Documentação

No diretório /pub/www/doc há vários artigos, preprints e guias na web.

Consulte a bibliografia online da WWW para obter uma lista destes e de outros artigos, livros, etc.

O arquivo /pub/www/doc/WWWBook.tar.Z é um extrato do texto da documentação do projeto WorldWideWeb (WWW).

Este é um instantâneo de um sistema de hipertexto em mudança. O texto é fornecido apenas como exemplo de hipertexto, não para distribuição geral. A exatidão de qualquer informação não é garantida, e nenhuma responsabilidade será aceita pelos autores por qualquer perda ou dano devido a imprecisão ou omissão. Uma cópia da documentação está inevitavelmente desatualizada e pode ser inconsistente. Existem links para informações que não são fornecidas nesse arquivo tar. Se algum desses fatos causar algum problema, você deve acessar os dados mestres originais pela rede usando www ou envie-nos um e-mail.

Servidores

O arquivo tar Daemon contém (nesta versão) o código para o daemon HTTP básico para servir arquivos e também para o gateway WWW-WAIS. Para compilar o gateway WAIS, você precisará de [um link para] uma distribuição WAIS no mesmo nível do diretório WWW.

Obtendo os arquivos WWW

Leia isto se você ainda não tiver os arquivos WWW necessários e não puder montar um diretório NFS no qual eles residem.

A fonte e o binário dos arquivos do projeto WWW estão atualmente disponíveis (veja copyright ) por FTP anônimo do nó info.cern.ch. Este nó é atualmente 128.141.201.74, mas ISSO PODE MUDAR. Os arquivos são archives de fita compactados (.tar.Z). Para obter um arquivo usando FTP anônimo, use o nome de usuário "anônimo" e seu endereço de e-mail como senha: -

> ftp info.cern.ch
Enter user name: anonymous
Password: me@myhost.edu
ftp> cd /pub/www/src
ftp> binary
ftp> get WWWLineMode_vvv.tar.Z
ftp> quit
>

O nome do arquivo dependerá do produto que você deseja. Aqui vvv é o número da versão do software.

Depois de ter o arquivo tar compactado, você deve descompactá-lo e descompactá-lo.


descompactar WWWLineMode_vvv.tar.Z

Isso criará um arquivo um pouco maior com o nome sem o ".Z". Para desembrulhar,

tar xvf WWWLineMode_vvv.tar


Agora você deve instalar o navegador de modo de linha , instalar o servidor , instalar o Viola , etc.

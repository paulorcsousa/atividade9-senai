Gerando HTML
Estes são scripts para gerar hipertexto SGML de coisas como listas de diretórios, etc. Além disso, para verificar e corrigir HTML duvidoso. Análise de log do servidor A análise dos logs do servidor requer, em primeiro lugar, a alteração dos números numéricos dos nós da Internet em nomes de domínio. httpd-analyse.c é um programa para fazer isso. Alimente os resultados através de awk e grep de sua escolha!
